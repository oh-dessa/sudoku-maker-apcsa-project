Title: Sudoku Maker! 

Name: Odessa Muñoz 

Desc: This program generates a randomized 9x9 Sudoku board!

Run: To run the project, just click run on github. 

Summary: This program makes a completed, solved 9x9 Sudoku Board. This follows key rules, such as each row, box, and column needing #s 1-9 without any duplicates. It starts in the main method, which calls the boardmaker() responsible for filling the board and checking for its validity using helper methods and then printing the board using boardprinter().
